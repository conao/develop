2. プログラムの説明
*********
#include<stdio.h>

main() {
    int a = 1;
    int b = 2, c;
    
    c = a > b || b + 1 != 1 & 5 == 5;
    printf("c=%d\n",c);
}

3. コメント
*********
bb9で示されたプログラムを入力し、動作を確認した。

c = a > b || b + 1 != 1 & 5 == 5;

を誤解のないように書くと、

c = ((a > b) || (((b + 1) != 1) & (5 == 5)));

となる。括弧の順に計算され、次のように推移する。

c = ((a > b) || (((b + 1) != 1) & (5 == 5)));
c = ((1 > 2) || (((2 + 1) != 1) & (5 == 5)));
c = (0 || (((2 + 1) != 1) & (5 == 5)));
c = (0 || ((3 != 1) & (5 == 5)));
c = (0 || (1 & (5 == 5)));
c = (0 || (1 & 1));
c = (0 || 1);
c = 1;

このため、標準出力に「c=1」が出力された。
