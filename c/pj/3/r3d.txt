プログラミング序説第3回レポート課題
B151235 山下直哉
課題番号 d

1. 端末での入出力
**********
b151235@imc2e-042:~/Develop/pj/3$ ./r3d
63.0

2. プログラムの説明
*********
// printf関数を使用するために必要
#include <stdio.h>

// int型を返すmain関数の宣言
int main() {
    // int型の変数の宣言
    int length;
    float basicWeight;

    // 変数targetへの代入
    length = 170;

    // 加減乗除は+-*/、また剰余は%で表現する.
    basicWeight = (length - 100) * 0.9;

    // printf関数. ""で囲まれた文字列を端末に出力する.
    // \nは改行コードを意味する.
    // %dはカンマの後ろで指定された変数で順番に置換される.
    // %dは整数型の指定であり、%fで浮動小数点型を指定する.
    // %<a>.<b>fと指定することで少数点以上a桁、小数点以下b桁の指定となる.
    // 桁溢れの場合、小数点以下b+1桁目が四捨五入され、丸められる.
    printf("%.1f\n", basicWeight);

    // main関数の正常終了
    return 0;
}

3. コメント
*********
問題の指定の通り、変数に一度代入した後、その値を使って計算し、
小数点以下1桁まで出力した.
入力は170で、「63.0」が出力された.
生の値は「63.000」であり、小数点以下第2位が四捨五入されていることがわかる.
